# Gricean Pragmatics

`gricean_pragmatics` is a project for evlauting LLM's pragmatic competence in multilingual settings. 

### Languages:

Supported languages include:

1. English
2. Modern Arabic
3. Japanese
4. Mandarin Chinese
5. Korean

Future supported languages include:

6. German
7. Spanish
8. Italian
9. Russian
10. French
